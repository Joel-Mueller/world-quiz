# Notes

Install the dependencies in package.json

```shell
npm install
```

Install new dependency and add it to package.json

```shell
npm install <package name>
```

Start a new node project

```shell
npm init -y
```

Start the server

```shell
npm start
```

Transpile TS code to JS

```shell
npx tsc
```

Start a js file

```shell
node dist/app.js
```
